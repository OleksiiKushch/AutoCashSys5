# Project

## Key technologies

- Core: Java 17, Spring Boot (WEB (MVC), Data, Test);
- Testing: JUnit, Mockito;
- Frontend: HTML, CSS, Bootstrap, JavaScript, AJAX, jQuery, Thymeleaf;
- Persistence: Spring Data JPA (Hibernate), MySQL, Liquibase;

### Database
